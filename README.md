# react-native-chunk-upload-files

this library sends the files in small chunks.

## Installation

```sh
npm install react-native-chunk-upload-files
```

## Usage


```js
import { multiply } from 'react-native-chunk-upload-files';

// ...

const result = multiply(3, 7);
```
